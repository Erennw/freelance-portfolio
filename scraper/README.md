# News Scraper

Automatically scrapes latest news from BBC and saves to Excel.

## Features
- Fetches news titles and links
- Exports data to Excel automatically
- Duplicate detection

## Technologies
- Python
- BeautifulSoup4
- Requests
- OpenPyXL

## Usage
```bash
pip install requests beautifulsoup4 openpyxl
python bbc_scraper.py
```
